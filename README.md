# Smart Helmet with ESP32 and React Capacitor Mobile App

## Overview
This project is a smart helmet system that integrates an ESP32 microcontroller with a React Capacitor mobile application for real-time monitoring and control.

## Features
- ESP32-based helmet control and data collection
- React Capacitor cross-platform mobile app
- Real-time wireless communication
- User-friendly interface

## Tech Stack
- **Backend/Hardware**: ESP32 microcontroller
- **Frontend**: React with Capacitor
- **Communication**: WiFi/Bluetooth

## Getting Started

### Hardware Requirements
- ESP32 development board
- Smart helmet components
- Wireless communication modules

### Mobile App
Built with React and Capacitor for iOS and Android compatibility.
